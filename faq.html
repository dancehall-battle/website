Frequently asked questions

What is the difference between an event and a battle?
A single event can have multiple battles. For example, Torino Dancehall Movement is one event with three battles: 1 vs 1, 2 vs 2, and "Who's that Gal?".
Which battles are included?
A battle has to fulfill all the following requirements to be included:
  • Contains 1 (one) or more rounds.
  • Contains 0 (zero) or more clash rounds.
  • Contains only freestyle rounds. Choreographic rounds are not allowed.
  • At least one judge is required.
  • The audience/public cannot be one of the judges.
